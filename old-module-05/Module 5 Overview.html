Module 5 Overview

In Module 5: Object Code - Control, you will continue to look at object code.

Topics covered will be:

  • Logical and Shift Operations
  • Using conditional codes to change the flow of instructions.
  • How gcc compiles some common C programming statements into assembly code.
  • These include if statements, for loops and switch statements.

Learning objectives

At the end of this learning module, you will be able to:

  1. Use logical and shift operations in assembly code using the gcc inline assembly language.
  2. Use conditional codes (jump instructions) in assembly code to change the flow of instructions.
  3. Implement if and switch statements and while loops using conditional codes in assembly code.

Learning activities

You are expected to do the following tasks in order to achieve the above objectives:

  1. Read Chapter 3, section 3.6 in the textbook
  2. Read Lessons 1 to 5 and complete the embedded practice problems
  3. Complete Discussion 5
  4. Take Quiz 5
  5. Review additional resources in the module summary
