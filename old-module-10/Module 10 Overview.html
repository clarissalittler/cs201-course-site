Module 10 Overview

In Module 10: Optimization, you will focus on how to optimize your code to run faster. There are other optimizations that are also important, such as memory or space optimization, but here we will concentrate on time optimization or performance.

  • How programs are compiled and executed
  • How to measure program performance and identify bottlenecks
  • How to improve performance without destroying code modularity and generality

Learning objectives

At the end of this learning module, you will be able to:

  1. Explore how to make programs run faster via several different types of program optimization.
  2. Interpret the capabilities and limitations of optimizing compilers.
  3. Describe a number of techniques for improving code performance.

Learning activities

You are expected to do the following tasks in order to achieve the above objectives:

  1. Read Chapter 5, sections 5.1 to 5.6, 5.8, 5.13, 5.14, and Chapter 6, sections 6.6 and 6.7 in textbook
  2. Read Lessons 1 to 4 and complete the embedded practice problems
  3. Complete Discussion 10
  4. Take Quiz 10
  5. Complete Assignment 4: Optimization
  6. Review additional resources in the module summary
