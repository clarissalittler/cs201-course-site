Wrap up

In this module, we looked at floating point numbers and studied how floating point is represented as binary numbers. Then we discussed how they are represented in computer memory. Other topics covered included rounding floating point numbers and problems encountered in doing floating point operations.

Key Takeaways

  • Fractional binary numbers have similar notation to decimal numbers. Basically, the fractional part is separated by a period.
  • IEEE Standard 754 Established in 1985 as a uniform standard for floating point arithmetic. Floating Point numbers are stored in a series of bit fields where each field is a part of the bit pattern. The fields represent parts of the floating point number when written in scientific notation.
  • All representations of floating point numbers have finite precision. There is only a fixed number of digits that can fit into the significand field. Since many floating point arithmetic operations end up with more digits than can fit into the significand, there is a need to drop off digits. This is called rounding.
