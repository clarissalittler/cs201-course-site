Module 2 Summary

Wrap up

We have studied integer data which are numbers without a decimal part. We have looked at the ranges (the maximum and minimum values) of each type, how they are encoded in memory, typecasting and overflow problems. These are common problems that a software engineer may encounter when working with integer numbers.

Key takeaways

  • Integer representation is of two types - Signed and Unsigned
  • Signed integers are encoded using Two's Complement
  • Overflow occurs when the full integer result cannot fit within the size limits of the data type
