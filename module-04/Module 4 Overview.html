Module 4 Overview

In Module 4: Introduction to Object Code, you will look at object code. Object code is any code produced at the compiling or assembling stages of building software. This code is something that most students probably have not seen in their development experience. 

Learning objectives

At the end of this learning module, you will be able to:

  1. Explain how to obtain assembly code from source code or from a program. 
  2. Interpret the general environment in which assembly code works in registers, program stacks, etc.
  3. Use the most common assembly code instruction: the mov instruction.
  4. Write code with the common arithmetic instructions used to perform integer math.

Learning activities

You are expected to do the following tasks in order to achieve the above objectives:

    1. Read Chapter 3, sections 3.1 to 3.5 in textbook
    2. Read Lessons 1 to 9 and complete the embedded practice problems
    3. Complete Discussion 4
    4. Take Quiz 4
    5. Complete Assignment 2: Object Code
    6. Review additional resources in the module summary
