Module 4 Summary

Wrap up

Object code is any code produced at the compiling or assembling stages of building software. This code is something that most students probably have not seen in their development experience (unless you've had experience working with this type of code).

Key Takeaways

  • How to obtain assembly code from source code or from a program. 
  • The general environment that assembly code works in: registers, program stacks, etc.
  • The most common assembly code instruction: the mov instruction.
  • The common arithmetic instructions used to perform integer math.
