_start Assembly Syntax Reference

This reference matches the style used in this course: GNU assembler (as), GNU linker (ld), hand-written x86-64 assembly, and direct entry at _start (no C runtime).

Minimal Program Skeleton

        .text
        .global _start
_start:
        mov $60, %rax      # syscall: exit
        xor %rdi, %rdi     # status 0
        syscall

Build and Run

$ as -o prog.o prog.s
$ ld -o prog prog.o
$ ./prog
$ echo $?

Multi-file link example:

$ as -o main.o main.s
$ as -o readInt.o readInt.s
$ as -o writeInt.o writeInt.s
$ ld -o app main.o readInt.o writeInt.o

Core Syntax Rules (AT&T)

  • Order is source, destination (not destination, source).
  • Immediates use $ (example: $42).
  • Registers use % (example: %rax).
  • Instruction size suffixes: b (1 byte), w (2), l (4), q (8).
  • 32-bit register writes (example: movl ... , %eax) zero-extend into the upper 32 bits of the 64-bit register.
  • # starts a comment.

Addressing Forms

General memory operand form:

displacement(base, index, scale)
  • Computed address = displacement + base + index*scale.
  • scale must be 1, 2, 4, or 8.
  • RIP-relative form for static data: label(%rip).
movq 16(%rsp), %rax
movq (%rbx,%rcx,8), %rdx
lea arr(%rip), %r8

Data Movement and Arithmetic

  • mov does not allow memory-to-memory operands in one instruction.
  • lea computes addresses/arithmetic; it does not read memory contents.
  • idiv uses the pair %rdx:%rax; run cqto before signed 64-bit division.
movq %rax, %rbx
movq $123, %rax
movq (%rsi), %rax
lea (%rdx,%rdx,4), %rdx      # rdx = rdx*5
addq %rbx, %rax
subq $8, %rsp
imulq $3, %rax, %rax
cqto
idivq %rcx

Compare, Test, and Jumps

cmpq %rbx, %rax      # flags from (rax - rbx)
testq %rax, %rax     # flags from (rax & rax)

Stack and Calls

pushq %rax
popq %rbx
call readInt
ret
  • pushq is equivalent to subq $8,%rsp then store at (%rsp).
  • popq loads from (%rsp) then adds 8 to %rsp.
  • Keep stack alignment consistent before call sites when interoperating with ABI-compliant routines.

Linux x86-64 Syscall Convention

FieldRegister
Syscall number%rax
Arg 1..6%rdi, %rsi, %rdx, %r10, %r8, %r9
Return value%rax
Clobbered by syscall%rcx, %r11

Common syscall numbers used in this course:

  • read = 0
  • write = 1
  • exit = 60

Common Patterns

Write a string to stdout

        .section .rodata
msg:    .asciz "Hello\n"
len = . - msg - 1

        .text
        .global _start
_start:
        mov $1, %rax
        mov $1, %rdi
        lea msg(%rip), %rsi
        mov $len, %rdx
        syscall

        mov $60, %rax
        xor %rdi, %rdi
        syscall

Exit with computed status

        mov $60, %rax
        mov %r12, %rdi
        syscall

echo $? reports the low 8 bits (0-255).

Common Mistakes Checklist

  • Using main instead of _start when linking directly with ld.
  • Forgetting .global _start.
  • Reversing operand order (AT&T is source then destination).
  • Forgetting $ on immediates.
  • Using invalid memory-to-memory arithmetic/move forms.
  • Using signed jump for unsigned compare (or vice versa).
  • Forgetting that syscall overwrites %rax with return value.
  • Failing to preserve needed registers/values across calls and syscalls.
