Process Lifecycle Syscalls

pid_t pid = fork();
if (pid == 0) {
    execl("/bin/ls", "ls", "-l", (char *)0);
    _exit(127);    // only runs if exec fails
}
int status;
waitpid(pid, &status, 0);
  • fork(): returns 0 in child, child PID in parent, -1 on error.
  • execve() family: replaces current process image, does not return on success.
  • waitpid(): reaps child and prevents zombies.
  • Use _exit() in child error paths after fork() if needed.

Zombies and Orphans

  • Zombie: child has exited but parent has not reaped it yet.
  • Orphan: parent exits first; child is re-parented (typically to PID 1/systemd).
  • Install SIGCHLD handling or call waitpid() reliably to avoid zombie buildup.

Installing Handlers (Recommended: sigaction)

volatile sig_atomic_t got_sigint = 0;

void on_sigint(int signo) {
    got_sigint = 1;
}

struct sigaction sa;
sa.sa_handler = on_sigint;
sigemptyset(&sa.sa_mask);
sa.sa_flags = 0;
sigaction(SIGINT, &sa, NULL);
  • signal() is simple but less precise across platforms; prefer sigaction().
  • Keep handler work minimal; set a flag and return.
  • Use volatile sig_atomic_t for shared handler flags.

Async-Signal Safety

  • Safe in handlers: low-level async-signal-safe functions such as write().
  • Unsafe in handlers: printf(), malloc(), most stdio and non-reentrant library code.
  • If output is needed inside a handler, prefer write(STDOUT_FILENO,...).

Blocking and Unblocking Signals

sigset_t mask, prev;
sigemptyset(&mask);
sigaddset(&mask, SIGINT);
sigprocmask(SIG_BLOCK, &mask, &prev);

// critical section

sigprocmask(SIG_SETMASK, &prev, NULL);
  • SIG_BLOCK: add to blocked set.
  • SIG_UNBLOCK: remove from blocked set.
  • SIG_SETMASK: replace blocked set exactly.

Process Groups and kill

  • kill(pid, sig): signal one process if pid > 0.
  • kill(0, sig): signal all processes in caller's process group.
  • kill(-pgid, sig): signal all processes in process group pgid.
$ kill -9 12345      # process 12345
$ kill -9 -15111     # process group 15111

Common Wait Macros

int status;

if (WIFEXITED(status)) {
    int code = WEXITSTATUS(status);

if (WIFSIGNALED(status)) {
    int sig = WTERMSIG(status);
}

Reliable SIGCHLD Reaping Loop

void on_sigchld(int signo) {
    int saved = errno;
    while (waitpid(-1, NULL, WNOHANG) > 0) {
    }
    errno = saved;
}

This pattern reaps all available terminated children and avoids leaving zombies.

Common Mistakes Checklist

  • Forgetting to reap children, causing zombies.
  • Calling non-async-signal-safe functions inside signal handlers.
  • Assuming blocked standard signals queue multiple times (they do not).
  • Ignoring return values and errno from fork/exec/waitpid/sigaction.
  • Using exit() in a post-fork child error path when _exit() is required.
