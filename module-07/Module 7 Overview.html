Module 7 Overview

In Module 7: Processes, we will start by discussing Flow Control and Exceptional Flow Control.  We will talk about exceptions, what causes them, and how they are handled. We also learn about Processes and how these processes affect Exceptions in a Linux system. 

Process control goes over the use of some popular library functions for creating and managing processes. We also look at several examples of using Fork to create child processes.

Learning objectives

At the end of this learning module, you will be able to:

  1. Explain about Flow Control and Exceptional Flow Control
  2. Interpret an Exception and how exceptions are handled
  3. Define the four classes of Exceptions
  4. Explain about the workings of Exceptions and Processes
  5. Demonstrate understanding of how to use Fork to create child processes

Learning activities

You are expected to do the following tasks in order to achieve the above objectives:

    1. Read Chapter 8, sections 8.1 to 8.4 in textbook
    2. Read Lessons 1 to 7 and complete the embedded practice problems
    3. Complete Discussion 7
