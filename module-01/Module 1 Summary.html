Wrap up

Compared to other high-level languages, C maps almost directly into hardware instructions, making code potentially more efficient. C also provides a minimal set of abstractions compared to other high-level languages. Data is everywhere, and this data includes numbers, symbols (letters, digits, etc), graphics, audio, and other data. In this course, we will concentrate on numeric data, different ways to represent the data, and how computers represent this data. 

Key Takeaways

  • C is prevalent in critical applications such as Operating systems (e.g., Windows, Linux, FreeBSD/OS X)‏, Web servers (apache, Google’s front-end)‏, Web browsers (Firefox, chrome)‏, Mail servers (sendmail, postfix)‏, etc.
  • For programmers, what really matters is how the finite word sizes used to represent integer and floating point data determine what values can be represented and the behavior of different operations
