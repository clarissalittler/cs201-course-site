Module 1 Overview

In Module 1: C Language and Numeric Representation, we will be covering the binary representation of data within a computer system. Data includes numbers, symbols (letters, digits, etc), graphics, audio, and other data. Throughout this course, we will concentrate on numeric data. We will also start learning the C language in this module.

Learning objectives

At the end of this learning module, you will be able to:

  1. Explain the basic syntax of C
  2. Demonstrate C Functions and Command Line Arguments
  3. Understand Bitwise Operators
  4. Explain Numeric Representation and the different numbering systems and how to convert from one to the other
  5. Interpret Byte Ordering, Bit Flags and Masking
