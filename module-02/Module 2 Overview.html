Module 2 Overview

In this module, you will learn the basics of integer representation and integer arithmetic.

Learning objectives

At the end of this learning module, you will be able to:

  1. Compare and contrast different representations of signed and unsigned integers using correct syntax
  2. Explain conversion between signed and unsigned integers
  3. Explain Integer Arithmetic and one of its consequences - Overflow

Learning activities

You are expected to do the following tasks in order to achieve the above objectives:

    1. Complete the book sections below
    2. Read Lessons 1-5 and complete the embedded practice problems
    3. Complete Discussion 2 with a post and 2 replies
    4. Take Quiz 2
    5. Begin Assignment 1: C Language and Integer Representation, due next week
    6. Review the Additional Resources on the Module 2 Summary page
