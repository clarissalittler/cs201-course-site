Integer Representation: Unsigned Integer Encoding

The encoding of how to represent unsigned int in memory is straightforward. The bit pattern is interpreted as a binary number which is then translated to a decimal when displayed to the screen.

For a given size w bits, the maximum value of an unsigned numeric type is given by the formula:

UMAX = 2w - 1 

The minimum value (UMIN) for unsigned is always 0.

Literal numbers (1, 0xa23, etc) are always interpreted as signed unless otherwise indicated as unsigned. If you need to make it clear that a literal is to be treated as an unsigned number, use u or U after the number to indicate unsigned literal.

127556u or 0x2A234U
