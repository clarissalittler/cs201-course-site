Module 3 Overview

In Module 3: Floating Point, you will look at floating point numbers. We will start by looking at how floating point is represented as binary numbers. Then we will discuss how they are represented in computer memory. Other topics covered are rounding floating point numbers and problems encountered in doing floating point operations.

In C/C++, there are only two commonly used floating point types: float, which is 32 bit, and double, which is 64 bit. The float type is considered to be single-precision, and the double type is considered to be double-precision. Precision is how many digits there are in a floating point number; the more digits, the higher the precision.

Learning objectives

At the end of this learning module, you will be able to:

  1. Represent fractional binary numbers
  2. Define the IEEE floating point standard, and identify examples and properties
  3. Do rounding, addition and multiplication of floating point numbers
  4. Identify floating point in C and how to use floats and doubles
