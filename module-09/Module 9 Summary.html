Module 9 Summary

Wrap up

In this module, we looked at Memory Hierarchy. The memory hierarchy includes internal main memory, CPU chip memory (registers and memory caches), and disk storage.

Key Takeaways

  • Internal main memory is RAM. RAM comes in two varieties - SRAM and DRAM.
  • DRAM and SRAM are volatile memories - Lose information if powered off. ROM, PROM, EPROM etc are examples of Nonvolatile memories that retain value even if powered off.
  • Well-written programs exhibit a property called locality.
  • Principle of Locality: Programs tend to use data and instructions with addresses near or equal to those they have used recently.
  • Being able to look at code and get a qualitative sense of its locality is a key skill for a professional programmer.
  • Cache: A smaller, faster storage device that acts as a staging area for a subset of the data in a larger, slower device.
  • Memory hierarchies based on caching close the gap by exploiting locality.
